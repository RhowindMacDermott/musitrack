# MusiTrack - Your Personal Music Journey

A modern web application that connects with your Spotify account to provide personalized music recommendations and insights about your listening habits.

## Live Demo

Visit [MusiTrack](https://rhowindmacdermott.github.io/musitrack/) to try the app!

## Features

- Secure Spotify OAuth integration
- View your Spotify profile
- Get personalized music recommendations
- Modern, animated UI with glass-morphism effects
- Responsive design that works on all devices

## Tech Stack

- HTML5
- Tailwind CSS for styling
- Vanilla JavaScript
- Spotify Web API
- Modern animations and transitions

## Setup for Development

1. Clone the repository
2. Open index.html in your browser
3. Connect with your Spotify account
4. Enjoy your personalized music recommendations!

## Spotify Integration

The app uses Spotify's OAuth 2.0 flow for authentication and accesses the following Spotify APIs:
- User Profile
- Top Tracks
- Recommendations
